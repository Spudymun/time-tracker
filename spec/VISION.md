# Time Tracker — Vision

> Создан: 2026-02-22
> Последнее обновление: 2026-02-22

## Elevator Pitch

Time Tracker — минималистичный веб-инструмент для учёта рабочего времени, вдохновлённый Toggl.
Позволяет фрилансерам трекать время по задачам и проектам, отмечать billable-часы,
группировать по тегам и получать отчёты за любой период с экспортом в CSV.

## Целевая аудитория

**Основной пользователь:** Разработчик или дизайнер-фрилансер, ведущий 3–10 проектов
одновременно. Хочет знать куда уходит время, сколько billable часов выставлять клиенту,
и быстро формировать еженедельный отчёт.

**Ключевые сценарии использования:**

1. **Дневной трекинг** — запустить таймер в начале работы, остановить по окончании.
   Задача привязана к проекту, помечена тегами и billable-флагом.
2. **Коррекция записей** — добавить/отредактировать пропущенную запись вручную
   (забыл запустить таймер — ввожу время руками).
3. **Еженедельный отчёт** — выбрать период, увидеть сводку по проектам, скачать CSV.
4. **Быстрое продолжение** — нажать «Continue» на прошлой записи — стартует новая
   с теми же параметрами.

## MVP Границы

**ВХОДИТ в MVP:**

- [x] Аутентификация: регистрация (email+password), вход (email+password, GitHub, Google), выход
- [x] Активный таймер: Start/Stop, описание с автодополнением, выбор проекта, billable-флаг, теги
- [x] Таймер тикает в хедере на всех страницах
- [x] Список записей с группировкой по дате и по проекту
- [x] Фильтрация записей по проекту, тегу, billable, текстовому поиску
- [x] Редактирование / удаление записей, ручной ввод hh:mm
- [x] Кнопка «Continue» на прошлых записях
- [x] Страница проектов: CRUD, цвет HEX, статистика часов
- [x] Архивация проектов (скрыть из выбора, но сохранить записи и отчёты)
- [x] Бюджет времени на проект (estimatedHours + прогресс-бар)
- [x] Почасовая ставка проекта (hourlyRate) — расчёт earnings в отчётах и Dashboard
- [x] Страница тегов: CRUD, цвет HEX
- [x] Страница отчётов: день/неделя/месяц/произвольный, двойное представление (by projects + by tags), CSV экспорт
- [x] Dashboard: bar chart по дням за неделю + breakdown по проектам + earnings суммари + возможная weekly target

## Метрики успеха MVP

- Пользователь запускает первый таймер за < 10 секунд после открытия
- Страница загружается < 2 секунд на Vercel
- API response < 300ms (p95)
- Весь обязательный функционал из тестового задания реализован

## Технические ограничения

- Multi-user: каждый пользователь видит только свои данные (изоляция по userId)
- Desktop-first (min-width: 768px), Chrome/Firefox/Safari
- PostgreSQL (Neon.tech serverless)
- Деплой на Vercel

## Нефункциональные требования

- **Производительность:** API < 300ms p95; таймер без janks (setInterval 1s)
- **Безопасность:** Zod-валидация всех входящих данных; никаких raw SQL queries
- **Доступность:** Tab/Space/Enter для основных действий
- **UX:** Toast-нотификации, loading states на кнопках
